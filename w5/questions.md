2521 Practice Midsemester Exam:
- [LINK](http://www.cse.unsw.edu.au/~cs2521/19t1/labs/sample-wk06-exam/index.html) If anything, please complete this first.

Linked List Problems:
- Assorted [problems](http://cslibrary.stanford.edu/105/LinkedListProblems.pdf) (with solutions!) from Stanford University. All you need to study for LinkedList problems
    - I recommend (at least) the problems: GetNth, InsertNth, Append, FrontBackSplit, RemoveDuplicates, Push, Reverse

Assorted Array Problems:
- [PROBLEM](https://leetcode.com/problems/merge-sorted-array/) Given two sorted arrays, merge them to produce another sorted array.
- [PROBLEM](https://leetcode.com/problems/maximum-product-of-three-numbers/) Given an array of N numbers, print out the maximum product of three numbers in that array.
- [PROBLEM](https://leetcode.com/problems/find-all-duplicates-in-an-array/) Print all duplicates in an array. (Extra Challenge: Do it in O(N) with O(1) space)
- [PROBLEM](https://leetcode.com/problems/transpose-matrix/) Transpose an NxM matrix.
- (Harder) [PROBLEM](https://leetcode.com/problems/spiral-matrix/) Given an NxM matrix print out the elements in spiral order.

Binary Tree Problems:
- [PROBLEM](https://leetcode.com/problems/construct-binary-search-tree-from-preorder-traversal/) Construct a binary search tree from preorder traversal
- [PROBLEM](https://leetcode.com/problems/same-tree/) Given two binary trees, check if they're identical.
- [PROBLEM](https://leetcode.com/problems/maximum-depth-of-binary-tree/) Compute the maximum depth of a binary tree.
- [PROBLEM](https://leetcode.com/problems/path-sum/) Determine if the sum of nodes in a path down the tree has a specific sum.
- [PROBLEM](https://leetcode.com/problems/path-sum-ii/) Print all the paths where the sum of the nodes add up to a specific sum.
- [PROBLEM](https://leetcode.com/problems/invert-binary-tree/) Invert a binary tree
